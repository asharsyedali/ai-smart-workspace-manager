# AI Smart Workspace and Meeting Manager

Final Year Project – BS Artificial Intelligence  
University of Management and Technology (UMT)

## Description
A bilingual (English–Urdu) AI-powered workplace system for:
- Meeting transcription
- Summarization
- Task extraction
- Emotion detection
- Attendance management
- Task delay prediction

## Tech Stack
- FastAPI
- React.js
- PostgreSQL
- Whisper, Wav2Vec2
- BERT, T5, XGBoost
- AWS
